# 🚀 Shop Billing System (Python-Based)

A simple command-line billing system built using Python to manage shop sales and generate bills.

---

## 🧠 Features
- Add products with price and quantity  
- Automatic bill calculation  
- Save bills as text files  
- View previously saved bills  

---

## 🛠️ Tech Stack
- Python  
- File Handling  
- Basic Data Structures (Lists)  

---

## ⚙️ How It Works
1. Add products to the cart  
2. Generate and save bill  
3. View saved bills anytime  

---

## 📁 Project Structure
- bills/ → Stores generated bills  
- main.py → Main program file  

---

## ▶️ Run the Project
```bash
python main.py
