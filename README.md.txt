# 🧾 Billing System (Python)

A simple and beginner-friendly **Billing System** built using Python.
This project allows users to add products, generate bills, and store them for future reference.

---

## 🚀 Features

✅ Add multiple products to cart
✅ Generate bill with total calculation
✅ Automatically save bills in files
✅ View previously saved bills

---

## 📂 Project Structure

```
Billing-System/
│
├── main.py        # Main program
├── bills/         # Stored bill files
└── README.md      # Project documentation
```

---

## ▶️ How to Run

Make sure Python is installed, then run:

```bash
python main.py
```
